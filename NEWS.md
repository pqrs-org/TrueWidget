# Changelog

## beta

-   💥 Breaking changes
    -   macOS 11 and macOS 12 are no longer supported.
-   ✨ New Features
    -   Added a compact display mode.
    -   Added a feature to display the other application's version.
-   ⚡️ Improvements
    -   Migrated to the SwiftUI life cycle.
    -   Sparkle Framework has been updated.
